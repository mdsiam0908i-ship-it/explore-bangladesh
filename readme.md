# Explore Bangladesh 🇧🇩

Explore Bangladesh is a comprehensive travel companion app designed for both local citizens and international tourists.

## Key Features
- **Smart Route Planner:** Plan your journey across 64 districts.
- **Emergency SOS & Hospital Locator:** Integrated safety features for travelers.
- **Regional Translator:** Communicate easily in local dialects.
- **Merchant & Guide Management:** A robust ecosystem for local businesses and tour guides.
- **Admin Control Panel:** Complete oversight of users, listings, and payments.

## Deployment
This project is optimized for deployment via:
- GitHub Pages
- WebIntoApp (for APK generation)
- Replit

## Technology Stack
- HTML5, CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- Progressive Web App (PWA) configuration
